Estrutura HTML

<!DOCTYPE html>
<!DOCTYPE html> declara a versão do HTML utilizada no documento, é importante para a correta renderização do conteúdo e é colocado no início do código HTML.
<html>
A tag <html> é utilizada para delimitar o início e o fim do documento HTML, ou seja, ela envolve todo o conteúdo do documento. É a partir dessa tag que são incluídas outras tags.
<head>
A tag <head> é uma das principais tags utilizadas em documentos HTML e é usada para incluir metadados do documento, como título da página, links para arquivos CSS e Javascript e outras informações que não são exibidas diretamente na página web.
<body>
A tag <body> é usada em documentos HTML para envolver o conteúdo que será exibido na página web, como textos, imagens, vídeos, links, entre outros elementos. É dentro da tag <body> que é definido todo conteúdo visível da página.
